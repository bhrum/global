# 网络测试与数据传输工具

## 概述

这是一个用于合法网络测试和数据分析的工具，旨在帮助网络管理员和开发人员评估网络性能、测试数据传输能力和进行系统压力测试。

## 主要功能

- **网络带宽测试**: 评估网络传输能力和带宽利用率
- **数据传输性能评估**: 测试数据包传输效率和丢包率
- **系统压力测试**: 评估系统在高负载下的性能表现
- **网络协议研究**: 用于学习和研究网络通信协议

## 技术特性

- **速率控制**: 内置带宽限制和发送间隔控制，避免网络过载
- **并发控制**: 限制并发线程数量，减少系统资源占用
- **数据包结构**: 使用合法的数据包格式，模拟正常网络流量
- **用户确认**: 要求用户明确确认使用目的和权限
- **自动停止**: 设定测试持续时间，自动停止测试

## 配置参数

```python
# 网络参数
CHUNK_SIZE = 512  # 数据块大小 (字节)
WORKER_COUNT = 5  # 并发线程数量
RATE_LIMIT_PACKETS_PER_SEC = 100  # 速率限制 (包/秒)
SEND_INTERVAL = 0.01  # 发送间隔 (秒)
MAX_BANDWIDTH_BYTES_PER_SEC = 1MB  # 最大带宽限制

# 测试参数
TEST_DURATION = 300  # 测试持续时间 (秒)
```
